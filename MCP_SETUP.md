## Что было сделано

✅ Интегрирован в основное приложение
✅ Обновлена функция веб-поиска для использования MCP

## Настройка API ключа

1. **Получите API ключ Tavily:**
   - Перейдите на https://tavily.com/
   - Зарегистрируйтесь и получите API ключ

2. **Создайте переменные окружения:**
   ```bash
   # В папке mcp-server создайте файл .env
   echo "TAVILY_API_KEY=ваш_реальный_ключ_tavily" > mcp-server/.env
   echo "MCP_PORT=8002" >> mcp-server/.env
   ```

## Запуск

### Вариант 1: Полный запуск (рекомендуется)
```bash
npm run dev:full
```
Это запустит:
- Базу данных
- Основной сервер (порт 80)
- Vite dev server (порт 8081)
- MCP сервер Tavily (порт 8002)

### Вариант 2: Ручной запуск
```bash
# В первом терминале - основной сервер
npm run server

# Во втором терминале - Vite
npm run dev

# В третьем терминале - MCP сервер
npm run mcp
```

```

```

## Возможные проблемы

1. **MCP сервер не запускается:**
   - Проверьте, что установлен TAVILY_API_KEY
   - Убедитесь, что порт 8002 свободен

2. **Ошибка "MCP search service unavailable":**
   - MCP сервер не запущен
   - Запустите `npm run mcp`

3. **Нет результатов поиска:**
   - Проверьте API ключ Tavily
   - Убедитесь, что у вас есть доступ к Tavily API
