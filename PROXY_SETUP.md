# Настройка прокси

## Важно! Проблема с OpenAI и прокси

**OpenAI блокирует запросы из определенных стран/регионов.** Если вы получаете ошибку:
```
403 {"error":{"code":"unsupported_country_region_territory","message":"Country, region, or territory not supported"}}
```

Это означает, что IP адрес вашего прокси заблокирован OpenAI.

## Решение проблемы

### Вариант 1: Отключить прокси для OpenAI (рекомендуется)
Не используйте прокси для запросов к OpenAI API, так как ваш серверный IP может быть разрешен.

```bash
# На сервере установите ТОЛЬКО OpenAI ключ, без прокси
export OPENAI_API_KEY="sk-proj-ваш_ключ_здесь"
# НЕ устанавливайте PROXY_URL для OpenAI
```

### Вариант 2: Использовать другой прокси
Если нужно прокси для других API (DuckDuckGo, Wikipedia и т.д.), используйте другой прокси-сервер.

```bash
export OPENAI_API_KEY="sk-proj-ваш_ключ_здесь"
export PROXY_URL="http://другой_прокси_сервер"
```

### Вариант 3: Прокси только для определенных API
Текущая реализация позволяет гибко настраивать прокси. Если PROXY_URL не установлена, прокси не используется.

## Текущая конфигурация

### Переменные окружения

```bash
# Обязательно
export OPENAI_API_KEY="sk-proj-ваш_ключ_здесь"

# Опционально (для других API)
export PROXY_URL="http://username:password@host:port"
```

### Статус прокси
При запуске сервера вы увидите:
```
🚀 Server starting with proxy: ENABLED/DISABLED
📡 Proxy URL: http://... or Not set
```

### API с поддержкой прокси
- ✅ OpenAI API (только если прокси настроен)
- ✅ CoinGecko API
- ✅ DuckDuckGo API (поиск в интернете)
- ✅ Wikipedia API (русский и английский)
- ✅ Glosbe API (определения и переводы)
- ✅ NewsAPI (новости)
- ✅ Stack Exchange API (технические вопросы)

## Диагностика

Если проблема сохраняется:
1. Проверьте логи сервера на наличие "proxy: ENABLED"
2. Попробуйте временно убрать PROXY_URL и протестировать
3. Убедитесь, что OpenAI ключ активен и имеет достаточный баланс

## Безопасность
- Никогда не коммитите API ключи в git
- Используйте HTTPS для всех соединений
- Проверяйте надежность прокси-серверов
